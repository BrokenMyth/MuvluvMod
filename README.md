# MuvluvMod

## [English](README_EN.md)

本仓库的文件适用于 **Windows 平台 DMM Game Player 版本**的游戏客户端（マブラヴ・ガールズガーデンX）

> **⚠️ 已从 BepInEx 迁移到 MelonLoader**，使用户安装时不再需要从 BepInEx 官网下载 Unity 补丁。

---

## 功能特性

-   **剧情文本翻译**：支持主线与活动角色剧情的中文翻译（本地文件优先，CDN 在线回退）
-   **人名/队名/标题翻译**：角色名、队伍名、剧情标题和副标题的中文化
-   **剧情历史翻译**：回看已读剧情时同步显示翻译文本
-   **选项翻译**：剧情中的选择分支同步翻译
-   **中文字体替换**：使用更纱黑体（Sarasa Gothic SC Bold）替换游戏默认日文字体，支持描边效果
-   **动态马赛克移除**：去除游戏内的剧情动态马赛克效果（可恢复）
-   **始终启用跳过按钮**：强制跳过按钮始终可用
-   **剧情语音不中断**：播放下一句台词时不中断当前语音
-   **自动跳过战斗**：自动点击跳过按钮跳过战斗场景
-   **禁用白屏闪光**：禁用剧本演出中的白屏/黑屏闪烁效果（LightFlash / DarkFlash）

## 使用方法

### 1. 准备工作

-   确保已安装游戏的客户端（DMM Game Player 版）
-   确认游戏可执行文件 `muv_luv_girlsgardenx_cl.exe` 所在目录
-   需要 **.NET 6 Runtime**（通常随 MelonLoader 附带，无需单独安装）

### 2. 下载插件

-   前往 [Releases 页面](https://github.com/anosu/MuvluvMod/releases) 下载最新版本（带绿色 `Latest` 标记）
-   展开 `Assets`，下载 `MuvluvMod.7z`（不要下载 `Source code`，那是源码）

### 3. 安装插件

-   将压缩包解压，得到 `version.dll`、`MelonLoader` 文件夹、`Mods` 文件夹等内容
-   将它们复制到与 `muv_luv_girlsgardenx_cl.exe` 相同的目录下
-   若已存在旧版本，可以先删除或直接覆盖

### 4. 启动游戏

-   正常启动游戏：从 DMM Game Player 启动或者从第三方 DMM 启动器启动，**而不是直接双击 `muv_luv_girlsgardenx_cl.exe` 启动！！！**
-   第一次启动时 MelonLoader 会自动生成必要的 IL2CPP 互操作程序集和配置文件（约需 10-30 秒）

### 5. 配置文件

-   首次运行后会在 `UserData` 文件夹生成：
    -   `Loader.cfg`（MelonLoader 加载器配置）
    -   `MuvluvMod.cfg`（插件配置，含 General 和 Translation 两个分类）
-   修改配置后需重启游戏生效
-   如需隐藏控制台窗口，请在 `UserData\Loader.cfg` 中将 `[console]` 下的 `hide_console` 设置为 `true`

### 6. 翻译数据

翻译数据支持两种加载方式，优先级从高到低：

1. **本地文件**（推荐离线使用）：将 `muvluvgg-translation` 翻译仓库克隆到游戏目录下，插件会自动从本地加载
2. **CDN 在线加载**：若本地文件不存在，自动从 GitHub Raw 获取翻译数据

首次遇到未翻译的剧本时，插件会自动将原文保存到本地翻译仓库的 `scenes_pending` 目录，方便后续翻译。

## 快捷键

-   `F2`：开启/关闭翻译
-   `F3`：开启/关闭始终启用跳过按钮
-   `F4`：开启/关闭语音中断（关闭后语音不会随台词切换而中断）
-   `F5`：开启/关闭自动跳过战斗
